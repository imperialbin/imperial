# API Documentation

IMPERIAL has a RESTful API that allows you to interact with IMPERIAL from virtually any application.

Heres a quick-start guide

IMPERIAL owns two domains

- [imperialb.in](https://imperialb.in) - The main domain
- [impb.in](https://impb.in) - The secondary domain

API endpoints are available on both domains.

- [api.imperialb.in](https://api.imperialb.in)
- [api.impb.in](https://api.impb.in)

## Responses

A response can look like this

```json
{
  "success": true,
  "data": {
    "test": "fields"
  }
}
```

Or if a failed request happened

```json
{
  "success": false,
  "error": {
    "message": "Bad Request",
    "errors"?: ZodErrors[]
  }
}
```

### Create a Document

Creating a document with IMPERIAL's API is really easy, you can either use IMPERIAL's JavaScript SDK, or you can interact with the API directly.

#### Using the SDK

<Tabs items={["js", "py", "rs"]}>
  <Tab>
    **JavaScript**: 
```js
import { Imperial } from "imperial.js";

const client = new Imperial(/_ token _/);
client.createDocument("hello world!").then(console.log, console.error);

````
  </Tab>
  <Tab>
    **Python**:
```python
from imperial import Imperial

imp = Imperial("imperial_000000000000000000000000")
document = imp.create_document("Hello, World!")
````

#### Using the API

<Tabs items={['JavaScript', 'Python', 'Rust']}>
  <Tab>
    **JavaScript**: 
```js
fetch("https://api.imperialb.in/v1/document", {
  method: "POST",
  body: JSON.stringify({
    content: "hello world!",
  }),
});
```
  </Tab>
  <Tab>
    **Python**:
```python
import requests

response = requests.post("https://api.imperialb.in/v1/document", json={
"content": "hello world!"
})

````

  </Tab>
  <Tab>
    **Rust**:
```rust
use reqwest::Client;

let client = Client::new();

let response = client.post("https://api.imperialb.in/v1/document")
    .json(&json!({
        "content": "hello world!"
    }))
    .send()
    .await?;
````

### Get a Document

Getting a document with IMPERIAL's API is really easy, you can either use IMPERIAL's JavaScript SDK, or you can interact with the API directly.

#### Using the SDK

<Tabs items={["js", "py", "rs"]}>
  <Tab>
    **JavaScript**: 
```js
import { Imperial } from "imperial.js";

const client = new Imperial(/_ token _/);
client.getDocument("document_id").then(console.log, console.error);

````
  </Tab>
  <Tab>
    **Python**:
```python

imp = Imperial("imperial_000000000000000000000000")
````
