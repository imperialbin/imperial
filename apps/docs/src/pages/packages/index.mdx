# IMPERIAL packages

IMPERIAL offers various packages so you can access the API efficiently and effectively. Here are the packages we offer:

- imperial.js: [docs](https://docs.impb.in/packages/javascript) [npm](https://www.npmjs.com/package/imperial.js) [github](https://github.com/imperialbin/imperial.js)
- imperial.py: [docs](https://docs.impb.in/packages/python) [pypi](https://pypi.org/project/imperial.py/) [github](https://github.com/imperialbin/imperial.py)
- imperial.rs: [docs](https://docs.impb.in/packages/rust) [crates.io](https://crates.io/crates/imperial.rs) [github](https://github.com/imperialbin/imperial.rs)

If you've made an unofficial/community package, you can submit it to us and we'll add it here.
